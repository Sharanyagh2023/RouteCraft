RouteCraft is a smart route optimization system designed to address common commuter issues such as app fatigue, high travel costs, and inefficient route planning in cities like Bengaluru. It simplifies the travel experience by reducing the need to switch between multiple applications and helps users make better commuting decisions.

The system provides intelligent travel suggestions by combining different modes of transport such as autos, metro, and walking into a single optimized journey. It compares real-time prices from services like Uber and Namma Yatri and recommends the best possible route based on multiple factors including cost, time, traffic conditions, weather, and even the user’s battery level.

RouteCraft includes several key features that enhance the commuting experience. It offers smart split-route optimization to save both time and money, along with real-time price comparison across multiple ride services. The system is weather-aware, avoiding walking routes during rain, and includes a battery-based “Survival Mode” that prioritizes faster routes when the device is low on power. Additionally, it supports one-click deep linking to booking apps and identifies intelligent pivot points for switching between transport modes efficiently.

The main goal of RouteCraft is to help users save money—typically more than ₹100 per trip—while reducing travel stress. By offering faster, smarter, and more reliable commuting options, it aims to improve the overall urban travel experience.

From a technical perspective, the system is built using FastAPI (Python) as the backend and integrates APIs such as Google Maps, OpenWeather, and various ride services. It is designed as a web-based Progressive Web App (PWA), ensuring accessibility without the need for installation. The platform focuses on high performance, scalability, and real-time decision-making to deliver a seamless user experience.
